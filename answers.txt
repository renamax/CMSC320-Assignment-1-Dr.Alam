Q1: Why might be consider using git?
Answer: We might consider using git because it is a very helpful version control tool for working on a coding project with multiple collaborators in real time without their modifications interfering with each other. 
Q2: What field could data science be applied to? Give and Example and how it might help.
Answer: One field data science could be applied to is healthcare. For example, data science could be used to aggregate health records of COVID-19 patients and predict the spread of the disease.
Q3: What is the difference between merging and rebasing?
Answer: Merging is a safe option that combines the histories of two branches into one, while rebasing rewrites the history by moving changes from one branch to another.
